# DragNDropTxtStringConcatenator
Concatenates strings in a .txt file. Drag and drop a file (MyFile.txt) on the .exe and it will output another file (MyFile.CONCAT.txt)

Turns this:
```
line1
line2
line3

line4
line5
line6

line7
line8
line9
```
Into this
```
line1line4line7
line2line5line8
line3line6line9
```
